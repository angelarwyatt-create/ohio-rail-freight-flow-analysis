# ohio-rail-freight-flow-analysis
This project analyzes inbound rail freight flows into Ohio using publicly available Freight Analysis Framework (FAF) data from the Bureau of Transportation Statistics.
### Key Questions
- What are the top commodities shipped into Ohio by rail?
- Which states are the primary origin points?
- How concentrated are freight volumes by commodity?

### Tools Used
- Tableau Public
- Microsoft Excel (FAF metadata mapping)
- FAF 5.7.1 public datasets
